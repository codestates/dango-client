# DANGO
 
<br>

### 당신 근처의 숨은 고수, 당고

DANGO는 지도를 기반으로 우리 동네 이웃들과 재능을 나눌 수 있는 서비스입니다.

<br></br>

DANGO는 누구나 재능을 가지고 있다는 생각에서 출발했습니다. 

당신의 숨겨진 재능을 이웃과 서로 나눠보세요!
